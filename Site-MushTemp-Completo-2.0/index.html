Quem Somos?

    Nós, alunos da SPTECH, estamos desenvolvendo um projeto voltado para o controle de temperatura e umidade em estufas de cogumelos. Este projeto envolve a implementação de um sistema de monitoramento por meio de sensores instalados nas estufas. Os sensores foram programados para fazer a coleta de ocilassão de temperatura, permitindo assim a coleta de informações precisas sobre as variações de temperatura nessas estruturas.

Objetivos:


• O projeto Mushtemp tem como objetivo desenvolver um sistema de controle de temperatura para estufas utilizadas no cultivo de cogumelos.


• Com o objetivo de garantir condições climáticas ideais para o crescimento e desenvolvimento saudável dos cogumelos A Mushtemp busca oferecer sensores de temperatura e umidade de forma eficiente para produtores e cultivadores, assim tendo maior lucro, otimizando o processo e diminuindo o desperdício.

Projeto MushTemp

Arduino

   Sensores instalados nas estufas de cogumelos irão captar dados como temperatura e umidade.
O Arduino, programado com código específico, receberá, processará e guardará no banco de dados os dados em tempo real.

Monitoramento

   Os gráficos mostrarão visualmente as oscilações de temperatura e umidade ao longo do tempo em cada estufa. Os gráficos serão acessíveis por meio de uma interface web, permitindo aos usuários monitorar as condições das estufas de forma intuitiva e eficaz.

Benefícios

Um controle preciso do ambiente para cultivo saudável, reduzindo custos ao minimizar perdas e aumentando os rendimentos. O monitoramento contínuo também previne doenças e pragas, contribuindo para uma cultura mais saudável.
